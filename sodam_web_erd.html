전체 관계 구조
실선은 NOT NULL FK(필수 참조), 점선은 nullable FK(선택 참조)를 의미합니다. 1 쪽이 PK 테이블, N 쪽이 FK 테이블입니다.
PK Primary Key
FK Foreign Key
NOT NULL FK
nullable FK
